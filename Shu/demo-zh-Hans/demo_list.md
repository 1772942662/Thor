## Shu 的使用技巧

Shu 现在是一个常用文件查看 + 轻量单线程 http 下载工具

查看器即 Viewer，所以并不是 Editor 或者 Manager，可以把 Shu 作为手机上的预览应用使用。

Shu 的长期目标是：万能文件查看器


### Shu 的 HTTP 下载

轻量单线程 HTTP 下载，不支持自动解析网页中的音视频资源
